# 💬​ Word Chain Game - 끝말잇기 게임
다수의 사람이 간단한 끝말잇기를 할 수 있는 게임 프로젝트를 진행하였습니다.

</br>

# 🎯 목표
- SQL을 이용한 데이터 베이스로 MFC와 연결하여 서버 구현
- 다수의 사람이 접속 가능한 끝말잇기 게임 구현
- 긴장감을 위한 타이머 기능과 배경음 추가
  
</br>
